How to Use
  1. Select First ROI Name - Choose an ROI structure name (e.g., "GTV", "CTV")
  2. Select Second ROI Name - Choose the same or different ROI name to compare
  3. Auto-Generate Pairs - System automatically creates all pairwise combinations
  4. Review Pairs - Check the generated comparison list below
  5. Compute Metrics - Click "Compute All Metrics" to calculate spatial overlap
Examples:
  • Same ROI, different structure sets: Select "GTV" → "GTV". If GTV exists in 4 structure sets, creates 6 pairs (1vs2, 1vs3, 1vs4, 2vs3, 2vs4, 3vs4)
  • STAPLE vs all instances: Select "GTV (STAPLE)" → "GTV". Creates STAPLE vs each instance (STAPLE vs 1, STAPLE vs 2, etc.)
  • Different ROIs: Select "GTV" → "CTV". Creates all combinations where both exist in same series
